# grok-150-java

Java practice repo for the **NeetCode 150** (which includes the Blind 75) problems,
organized by algorithm/pattern category, using Gradle and JUnit 5.

Categories follow the NeetCode roadmap: Arrays & Hashing, Two Pointers, Sliding Window,
Stack, Binary Search, Linked List, Trees, Tries, Heap / Priority Queue, Backtracking,
Graphs, Advanced Graphs, 1-D DP, 2-D DP, Greedy, Intervals, Math & Geometry, Bit Manipulation.

Most classes are blank placeholders that throw `UnsupportedOperationException` with a
ready (but `@Disabled`) JUnit test, so you can implement each problem and flip its test on.
See [REORG_NOTES.md](REORG_NOTES.md) for the full layout and status.

## Run tests

```bash
./gradlew test
```
